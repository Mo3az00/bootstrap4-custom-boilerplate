Custom Bootstrap 4 Boilerplate

Use this document as a way to quickly start any new project.
All you get is this text and a mostly barebones HTML document.

You can customize which Bootstrap features should be loaded,
by modifying the file /assets/src/scss/_bootstrap.scss.

If you want to override Bootstrap variables, freel free to use
the file /assets/src/scss/_bootstrap-variables.scss.

Font Awesome is already built-in!
